- 工具库

## 功能

- 辅助编写插件
- 提供通用数据结构与默认样式

## 如何使用

  1. `git clone https://github.com/wenxig/delta-comic-core`
  2. `bun run build`
  3. `pnpm i "/path/to/delta-comic-core"`
